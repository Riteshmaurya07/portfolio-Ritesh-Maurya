# Ritesh Maurya's Personal Portfolio Website 🚀

Welcome to my personal portfolio repository! This is a modern, responsive, and performance-optimized portfolio website showcasing my skills, education, projects, achievements, and professional experience.

🌐 **Live Link:** [Visit My Portfolio](https://portfolioriteshmaurya.vercel.app)

---

## 📌 Features

- 🌓 **Dark/Light Mode:** Dynamic styling variables mapped to a Sun/Moon theme toggle with persistence in `localStorage`.
- ⚡ **Flagship Project Highlight:** Featured visual emphasis on flagship projects like **CodeX Live**.
- 📊 **Dynamic GitHub Stats:** Real-time stats integration (Streak, Profile Stats, and Top Languages) that adapt query parameter styling based on the active theme, alongside a visual Contribution Graph.
- 📱 **Mobile Responsiveness:** Fully responsive design adapting to all screen sizes from mobile to ultra-wide displays.
- 🎨 **Rich UI/UX:** Floating animations, scroll reveals, particle backgrounds, and glassmorphism.
- 📬 **Interactive Contact Form:** Secure client-side email delivery using EmailJS API integration.
- 📝 **Clean Architecture:** Structured components fetching project and skill representations dynamically via standard `JSON` databases.

---

## 🛠️ Tech Stack

- **Structure:** HTML5, Semantic Markup
- **Styling:** CSS3, Custom Styling Variables
- **Interactivity:** JavaScript, jQuery, Typed.js, Particle.js, Vanilla-Tilt.js, ScrollReveal
- **Integrations:** EmailJS API, GitHub Readme Stats API, GitHub Activity Graph API

## 💻 Local Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Riteshmaurya07/Portfolio-Website.git
   ```
2. **Navigate into the folder:**
   ```bash
   cd Portfolio-Website
   ```
3. **Open `index.html`** in your favorite browser, or serve it using Live Server in VS Code.
